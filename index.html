Javazon Lightning — Guide complet (du Niveau 45 → à la fin du jeu)

Page one-shot prête à être poussée sur GitHub. Contient paliers, zones de butin, recettes d'artisanat et estimations de chance de butin.

Résumé rapide

Progression: Niveau 45 → Fin du jeu

Suivre les paliers dans l'ordre, farmer les runes clés via la **Comtesse**, les **Supercoffres de Kurast Inférieur** et **Travincal**, prioriser la **Vengeance des Titans**, **Esprit**, **Queue de Rasoir** puis **Infinité** et **Énigme**.

Palier 1 — Niveau 45 → 65

But : préparer le **mode Cauchemar** → entrer dans le **mode Enfer**

Vengeance des Titans (Javelot Unique)

Effets : +2 Compétences Amazone / +2 Compétences Javelot / Vol de vie

Où : Mephisto **Mode Cauchemar**, Baal **Mode Cauchemar**. Chance estimée : très faible — plusieurs centaines de parcours possibles.

Informations sur l'artisanat / le butin

C'est un unique donc non réalisable. Farm Mephisto en boucle (Cauchemar → Enfer) et parcours de Baal.

Esprit (Monarque) — Mot Runique

Effets : +2 Compétences, **Récupération Rapide après coup**, Mana, Résistances

Où : Artisanat (Tal+Thul+Ort+Amn) — Farm la **Comtesse** pour les runes. Chance d'obtenir les runes : moyenne (Ort/Amn fréquentes, Tal/Thul moins).

Trahison (Mot Runique)

Effets : **Vitesse d'Attaque Accrue**, Estompe (résistances)

Où : Artisanat (Shael+Thul+Lem) — Lem est la rune la plus coûteuse ici (longues sessions de farm).

Queue de Rasoir (Ceinture)

Effets : 100% Pénétration

Où : Baal **Mode Cauchemar** / Mephisto **Mode Enfer**. Chance : rare — butin de boss unique.

Palier 2 — Niveau 65 → 85

But : farm du **mode Enfer**, viser les Mots Runiques de haut niveau

Vengeance des Titans éthérée (amélioration)

Où : Mephisto **Mode Enfer** / Baal **Mode Enfer** / **Fosses**. Chance : très faible → beaucoup de *grind* (répétition).

Chaînes d'Honneur (Mot Runique)

Où : Artisanat (Dol+Um+Ber+Ist). **Ber** et **Um** sont rares ; utiliser les **Supercoffres de Kurast Inférieur** / **Travincal** pour les runes.

Œil de Griffon

Où : Butin de boss en **Mode Enfer** (Mephisto / Baal / Pindle). Chance : très basse.

Colère du Grand Seigneur (Amulette)

Où : Diablo / Baal / butin de la fin du jeu. Chance : rare.

Palier 3 — Fin du jeu (Niveau 85+)

But : farm absolu — **Infinité**, **Énigme**, **Phénix**

Infinité (Mot Runique — Arme du Mercenaire)

Runes: Ber + Mal + Ber + Ist. Où farm : **Supercoffres de Kurast Inférieur**, Travincal, Chaos. Chance : très faible (les Supercoffres augmentent la probabilité).

Énigme (Mot Runique)

Runes: Jah + Ith + Ber. Où farm : Kurast Inférieur / Travincal / Chaos. Chance : extrêmement faible (Jah + Ber très rares).

Phénix (Mot Runique)

Runes: Vex + Vex + Lo + Jah. Où : boss du mode Enfer + Kurast Inférieur. Chance : extrêmement faible.

Roadmap d'obtention (ordre conseillé)

Suivre cet ordre pour limiter le temps perdu
  1. **Comtesse en Mode Cauchemar** → **Esprit**, **Trahison**, **Tradition** (si nécessaire) et runes de base.
  2. Mephisto en Mode Cauchemar → Vengeance des Titans, Couronne du Paysan, Queue de Rasoir.
  3. Baal en Mode Cauchemar → Expérience + Vengeance des Titans + équipement.
  4. **Comtesse en Mode Enfer** → farm les runes **Ist**.
  5. Fosses en Mode Enfer / Vaches en Mode Enfer → Uniques et bases.
  6. **Supercoffres de Kurast Inférieur** → farm les runes Vex, Jah, Ber.
  7. Artisanat d'Infinité → priorité si possible (pour le Mercenaire).
  8. Artisanat d'Énigme / Phénix → finaliser l'équipement.
